# TaskFlow_Telegram 🤖

Telegram-бот для управления задачами, трекинга времени и командной работы с детальной аналитикой.

---

## 📌 Возможности

### 🎯 Управление задачами
- ➕ **Создание задач** — с указанием дедлайна, категории, приоритета (🔴 Высокий / 🟡 Средний / 🟢 Низкий)
- 📋 **Подзадачи** — разбивайте большие задачи на подзадачи с отслеживанием прогресса
- 🔄 **Повторяющиеся задачи** — ежедневные, еженедельные, ежемесячные (автосоздание)
- 📂 **Шаблоны задач** — сохраняйте часто используемые задачи
- 🏷️ **Категории** — группируйте задачи по типам деятельности
- 😊 **Эмодзи-реакции** — добавляйте эмоции к задачам (😊😍🤔😰🔥💪🎯⭐💡🚀)

### ⏱️ Трекинг времени
- Засекайте время выполнения задач
- Автоматический подсчёт затраченного времени
- Уведомления о слишком долгих сессиях (>2 часов)

### 📊 Статистика и аналитика
- Общая статистика по всем категориям
- Статистика за день / неделю / месяц
- **Сравнение периодов** — неделя vs неделя, месяц vs месяц, год vs год
- **Топ задач** — самые времязатратные задачи
- **Дневная активность** — график продуктивности
- **Прогнозы времени** — предсказание времени на основе истории
- **Экспорт в CSV** — выгрузка статистики для анализа

### 📦 Проекты и командная работа
- Создание проектов с участниками
- **Роли:** владелец, администратор, участник
- Общие задачи внутри проекта
- Комментарии к задачам
- Управление участниками (добавление, удаление, смена роли)

### 🔔 Уведомления
- ⏰ Напоминания о дедлайнах (за 1 час)
- ⚠️ Оповещения о просроченных задачах
- 📈 Напоминания о прогрессе (если задача застопорилась)
- 📅 Ежедневные сводки (настраиваемое время)
- 📅 Еженедельные сводки (настраиваемый день и время)

### 🏆 Достижения
- 10+ достижений для мотивации:
  - Первый шаг, Мастер задач, Хранитель времени
  - Ранняя пташка, Ночная сова
  - 7 дней подряд, 30 дней подряд
  - Лидер проекта, Комментатор
- Автоматическая разблокировка при выполнении условий

### 🎓 Интерактивные туторы
- Пошаговое обучение для новых пользователей
- Закрепление навыков через практику

## 🚀 Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/TechnoSwey/TaskFlow_Telegram-Bot.git
cd TaskFlow_Telegram-Bot
```

2. Установка зависимостей

```bash
pip install -r requirements.txt
```

3. Настройка переменных окружения

Создайте файл .env в корневой папке:

```
BOT_TOKEN=ваш_токен_бота
DATABASE_NAME=tasks.db
```

Получить токен можно у @BotFather в Telegram.

4. Запуск бота

```bash
python run.py
```

Или через init_db.py для инициализации базы данных:

```bash
python init_db.py
python run.py
```

**📋 Пример использования**

Создание задачи

1. Нажмите "➕ Создать задачу"
2. Введите название (можно указать дедлайн в формате ГГГГ-ММ-ДД ЧЧ:ММ)
3. Выберите категорию и приоритет

Трекинг времени

1. Перейдите в раздел "⏱️ Таймер"
2. Выберите активную задачу
3. Нажмите "⏱️ Начать таймер"
4. По завершении нажмите "⏹️ Остановить таймер"

Работа с проектами

1. Создайте проект через "📦 Проекты"
2. Добавьте участников по username
3. Создавайте задачи внутри проекта
4. Управляйте ролями участников

Просмотр статистики

· Общая статистика по всем категориям
· Сравнение периодов
· Топ задач
· Прогнозы времени
· Экспорт в CSV

**🛠️ Технологии**

· Python 3.8+
· python-telegram-bot — работа с Telegram API
· SQLite — хранение данных
· APScheduler — планировщик уведомлений
· Matplotlib/Pandas — для графиков (опционально)

**📊 База данных**

Основные таблицы:

· users — пользователи
· categories — категории задач
· tasks — задачи (с приоритетами, подзадачами, повторениями)
· projects — проекты
· project_members — участники проектов
· task_comments — комментарии к задачам
· time_sessions — временные сессии
· achievements — достижения
· user_achievements — разблокированные достижения пользователей
· daily_activity — ежедневная активность
· predictions — прогнозы времени
· tutorial_progress — прогресс туториалов
