# TaskFlow_Telegram 🤖

Telegram-бот для управления задачами и трекинга времени с детальной аналитикой.

## 📌 Возможности

- ➕ **Создание задач** — с указанием дедлайна и категории
- 📂 **Шаблоны задач** — сохраняйте часто используемые задачи
- ⏱️ **Трекинг времени** — засекайте время выполнения задач
- 🏷️ **Категории** — группируйте задачи по типам деятельности
- 📊 **Статистика** — анализируйте время по категориям, дням, месяцам и годам
- 🔔 **Уведомления** — автоматические напоминания о дедлайнах
- 📅 **Ежедневные сводки** — отчёты о задачах за день (в 21:00)
- 📈 **Еженедельные сводки** — отчёты за неделю (по воскресеньям в 20:00)

## 🚀 Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/TechnoSwey/TaskFlow_Telegram.git
cd TaskFlow_Telegram
```

**2. Установка зависимостей**

```bash
pip install -r requirements.txt
```

**3. Настройка переменных окружения**

Создайте файл .env в корневой папке и добавьте токен вашего бота:

```
BOT_TOKEN=ваш_токен_бота
```

Получить токен можно у **@BotFather** в **Telegram**.

**4. Запуск бота**

```bash
python bot.py
```

Бот создаст базу данных tasks.db автоматически при первом запуске.

**📋 Пример использования**

**Создание задачи**

1. Нажмите "➕ Создать задачу"
2. Введите название задачи (можно указать дедлайн в формате ГГГГ-ММ-ДД ЧЧ:ММ)
3. Выберите категорию из списка или создайте новую

**Трекинг времени**

1. Перейдите в раздел "⏱️ Таймер"
2. Выберите активную задачу
3. Нажмите "⏱️ Начать таймер"
4. По завершении нажмите "⏹️ Остановить таймер"

**Просмотр статистики**

· Общая статистика по всем категориям
· Статистика за день/неделю/месяц
· Детальный отчёт по выбранной категории
· Анализ затраченного времени за год

**🛠️ Технологии**

· Python 3.8+
· python-telegram-bot — работа с Telegram API
· SQLite — хранение данных
· APScheduler — планировщик уведомлений
